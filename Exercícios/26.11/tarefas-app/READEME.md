# Aplicação de Lista de Tarefas

Uma aplicação simples de gerenciamento de tarefas construída com Next.js 15, TypeScript e testes unitários.

## Funcionalidades

- Listar tarefas
- Adicionar novas tarefas
- Contador de tarefas
- Testes unitários completos

## Tecnologias

- Next.js 15
- TypeScript
- Jest
- Testing Library
- Tailwind CSS (para estilização)
