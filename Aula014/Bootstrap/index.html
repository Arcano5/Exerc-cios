Por que personalizar o bootstrap
Motivações
1)Identidade visual: 2)Originalidade Personalização comuns: Cores Tamanhos de fontes Espaçamentos Bordas e sombras Como? 1) Sobreescrever classes via css exemplo: .btn-primary{ background-color:#xxxxxx; border-color: #yyyyyy; }

Usar Variaveis do Bootstrap (SCSS)
O bootstrap é escrito em SASS, o que permite personalizar editando variáveis antes da importação dos arquivos .scss
Precisaria ter instalado o bootstrap via npm
npm install bootstrap
@import"../node_modules/bootstrap/scss/bootstrap";
