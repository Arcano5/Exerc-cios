Tailwind CSS

O tailwind é semelhante ao bootstrap, porém é mais flexível para a costumização.

Como funciona?

Cada classe representa uma responsabilidade visual única


Por exemplo: bg-blue-500: fundo azul


text-center; mt-4: margem top 1rem


font-bold: texto em negrito

Como aplicar? o jeito mais fácil é via CDN (Rede de Entrega de Conteúdo)
